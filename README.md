## Endpoints

### 1. Get Current Time

- **Endpoint**: `/time`
- **Method**: `GET`
- **Description**: Retrieves the current time in the specified time zone.
----
- **Endpoint**: `/time/timezones`
- **Method**: `GET`
- **Description**: Retrieves the current time in the specified time zone.
